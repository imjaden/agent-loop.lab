# Agent Loop Prompt: Agent Loop 数据流可视化 HTML

## 任务目标

基于 `agent-loop.py` 的源码和 `README.md` 中的 Loop 架构图，生成一个**单页交互式可视化 HTML 文件**，展示 Agent Loop 从输入到输出的完整数据流。核心交互：**点击节点 → 百叶窗式收起可视化图 → 展开源码片段**。

---

## 页面架构

### 单页原则（强制）
- **必须**是单页 HTML，所有内容在一个文件中
- 不允许分页，除非满足以下条件：
  - 条件 A：源码片段过长（>500行），按逻辑模块拆分（给出理由）
  - 条件 B：可视化图与源码并行展示导致性能问题（给出理由）
- 理由格式：`<!-- [理由] -->`

### 页面布局（纵向流程图 + 百叶窗交互）

```
┌─────────────────────────────────────────────────────┐
│  Agent Loop · 数据流与源码映射                        │
├─────────────────────────────────────────────────────┤
│  [Legend: 入口 | Think | Act | Observe | Verify | 输出] │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │         可视化数据流图（默认展开）             │   │
│  │                                              │   │
│  │  入口 → Think → Act → Observe → Verify → 输出│   │
│  │         ↑              ↓                      │   │
│  │         └────── Loop ──┘                      │   │
│  │                                              │   │
│  └─────────────────────────────────────────────────────┘
│                                                        ← 点击节点时：
│  ┌──────────────────────────────────────────────────┐    可视化图收起
│  │  ▼ 入口 / main() 入口                        │    │   源码展开
│  ├──────────────────────────────────────────────────┤    │   (二者互斥，
│  │  [源码片段：main() 函数]                     │    │    百叶窗切换)
│  └──────────────────────────────────────────────────┘    │
│                                                        ←
│  ┌──────────────────────────────────────────────────┐
│  │  ▼ Think / LLM 分析                         │    │
│  ├──────────────────────────────────────────────────┤    │
│  │  [源码片段：LLM 调用]                        │    │
│  └──────────────────────────────────────────────────┘    │
│
│  ... 其他节点                                       │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 交互设计：百叶窗式切换

**默认状态**：可视化数据流图完全展开，节点列表收起

**点击节点**：
1. 可视化数据流图**百叶窗式收起**（向上收拢动画，max-height 渐变）
2. 同时节点详情**百叶窗式展开**（向下展开动画）
3. 展示该节点对应的源码片段（带语法高亮）

**再次点击**：
1. 源码片段**百叶窗式收起**
2. 可视化数据流图**百叶窗式展开**
3. 回到默认状态

**核心 CSS 动画**：
```css
.shutter {
  max-height: 0;
  overflow: hidden;
  transition: max-height 0.5s ease-out;
}
.shutter.open {
  max-height: 2000px;  /* 足够大的值 */
  transition: max-height 0.5s ease-in;
}
```

**替代方案**：如果百叶窗动画过于复杂，可采用：
- 淡入淡出：`opacity 0 → 1`，`transition: opacity 0.3s`
- 滑入滑出：`transform: translateY(-100%) → translateY(0)`

## Loop 执行步骤

### Step 1: 感知（Read）

读取 agent-loop.py 和 README.md，理解以下映射关系：

```
Flow 节点         对应的源码类/函数
──────           ────────────────
main() 入口      main() 函数（L659-776）
Think            Agent.run() 中的 LLM 调用（L392-416）
Act              工具执行循环（L454-476）
Observe          tool 结果注入 messages（L466-480）
Verify           Verifier 类 + 验证逻辑（L255-277, L420-451）
Budget 检查       Budget 类 + expired 判断（L284-303）
LoopState        状态持久化 save/load（L310-334）
最终输出         main() 中的输出格式 + 异常处理（L747-777）
```

### Step 2: 设计（Plan）

规划单页 HTML 布局：

```
单页结构：
├── <header>       标题 + Legend
├── <section.viz>  可视化数据流图（可百叶窗收起）
├── <section.nodes>
│   ├── <details.entry>   入口节点（可百叶窗展开源码）
│   ├── <details.think>   Think 节点
│   ├── <details.act>     Act 节点
│   ├── <details.observe> Observe 节点
│   ├── <details.verify>  Verify 节点
│   └── <details.output>  输出节点
└── <footer>       命名规范说明
```

### Step 3: 生成（Generate）

生成单文件 HTML，包含：

1. **纵向流程图**（默认展开，可百叶窗收起）：
   - 入口 / Think / Act / Observe / Verify / 输出 / 循环回退连线
   - 每个节点用颜色区分（入口蓝、Think紫、Act橙、Observe绿、Verify黄、输出蓝）
   - 可通过按钮或节点点击**百叶窗收起**，展开源码

2. **百叶窗交互**（核心交互）：
   - 点击节点标题 → 收起数据流图 + 展开源码片段
   - 再次点击 → 收起源码 + 展开数据流图
   - 使用 `max-height` + `transition` 实现百叶窗效果
   - 源码片段标注行号范围

3. **源码映射要求**：
   - 每个源码片段标注行号范围（如 `L659-776`）
   - 源码语法高亮（关键字/字符串/注释/函数名区分颜色）
   - 关键数据流用中文注释标注

4. **视觉效果**：
   - 暗色主题（#0b0e1a 背景）
   - 连接线带箭头
   - Legend 标签可点击跳转节点
   - 响应式布局

### Step 4: 验证（Verify）

对照以下检查清单验证输出：

```
□ 单页 HTML（除非给出必要理由，否则不分页）
□ 流程图包含 7 个节点：入口 / Think / Act / Observe / Verify / 输出 / 回退连线
□ 点击节点 → 百叶窗收起数据流图 + 展开源码
□ 再次点击 → 百叶窗收起源码 + 展开数据流图
□ 百叶窗动画流畅（max-height transition 或 opacity transition）
□ 源码片段标注行号范围
□ 语法高亮区分关键字/字符串/注释
□ 暗色主题一致
□ 响应式布局
□ 命名规范：AgentLoop数据流可视化-v1.0-{YYYYMMDD}.html
```

### Step 5: 停止条件
- ✅ 所有检查项验证通过 → 输出 HTML
- ❌ 任一检查项不通过 → 修复后重新验证（最多重试 3 次）
- ⏰ Token 预算或时间预算耗尽 → 输出当前状态
